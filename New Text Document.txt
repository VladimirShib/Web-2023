git clone (ссылка с сайта, code)
-клонирует репозиторий на комп

git add .
-подготовить к загрузке все файлы в этой папке (потому что точка). если нужен конкретный файл, то вместо точки писать название файла (например readme.txt)

git status
-показать какие файлы готовы к выгрузке (зеленые готовы, красные нет)

git commit -m "любой комментарий"
-добавляет комментарий к файлам, которые готовы к выгрузке (зеленые) (будут видны на гитхабе) обязательно! иначе не отравится (и не обновится)

git push
-отправить на гитхаб все подготовленные (зеленые) файлы вместе с комментариями

git pull
-скачивает в папку репозитория на компьютере все файлы с сайта
измененные файлы заменяет новой версией (перезаписывает)
новые файлы скачивает
если на сайте удален файл (другой разработчик удалил), то на компьютере этот файл тоже удалится
